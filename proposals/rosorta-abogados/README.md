# Propuesta demo · Rosorta-like

Demo estática para validar diseño de propuesta comercial tipo Rosorta con:

- navegación por secciones,
- bloques e info cards,
- espaciado consistente,
- impresión A4 sin desbordes.

## Vista local

```bash
python3 -m http.server 4173
# abrir http://localhost:4173/proposals/rosorta-abogados/
```

## Print

Usar el botón **Imprimir A4** o `Ctrl/Cmd + P`.
